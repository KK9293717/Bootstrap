Hello,World!

Bootstrap is an open-source and front end framework. This framework helps in faster web development and support developers in creating responsive web pages at a quicker pace. Bootstrap provides the basic structure of HTML and CSS with predefined classes, which help you to build functionality quickly and easily with device responsiveness. Thus, It saves plenty of time. For example, no need to start developing from scratch. It is used to build Mobile-first sites.

Heading

The first name of Bootstrap was Twitter Blueprint it was developed by Mark Otto and Jacob Thornton at Twitter. It got released in August 2011 on GitHub. Before the existence of Bootstrap, for making responsive sites and interface development, various external libraries were used, that not only brings inconsistency but also gave rise to the heavy maintenance burden.The bootstrap new version has come up with some significant changes, adding new components, eliminate others.
